# MVP Features

## Goal
Build a simple, functional prototype that demonstrates AI-powered matching and real-world connection.

---

## 🎯 Priority Features (Must Have)

### 1. Onboarding Quiz
- Owner personality questions
- Dog temperament and behavior questions
- Activity level and preferences

---

### 2. Compatibility Matching
- Basic scoring system (rule-based or AI-assisted)
- Matches users based on:
  - Personality similarity
  - Dog compatibility
  - Activity alignment

---

### 3. Match Explanation
- Display “Why you matched”
- Example:
  - “Both owners prefer low-pressure outdoor interaction”
  - “Both dogs have moderate energy and friendly temperament”

---

### 4. Simple UI
- Profile creation (owner + dog)
- Match display screen
- Clean and minimal design

---

### 5. Meetup Recommendation (Basic)
- Suggest general meetup idea:
  - “Try meeting at a local park”
  - “Best for low-energy dogs: short walk”

---

## 🚫 Not Needed for MVP
- Full messaging system
- Real-time GPS tracking
- Complex AI models
- Payment systems
- Full authentication system

---

## 🧠 AI Scope (Keep it Simple)
- Start with rule-based scoring
- Optionally enhance with LLM explanations
- Focus on clarity over complexity

---

## 🏁 Success Criteria
- User can complete onboarding
- User receives a match
- User sees explanation
- Demo is smooth and understandable

---

## ⚠️ Team Reminder
Do NOT overbuild.

Clarity > complexity  
Demo > perfection  
Story > features
