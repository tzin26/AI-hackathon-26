# PetPlayDate.ai SwiftUI Prototype

This folder contains a SwiftUI recreation of the multi-step `PetPlayDate.ai` flow:

1. Login
2. Sign Up
3. Profile Setup
4. Dog Details

## Suggested Xcode setup

1. Create a new iOS App project in Xcode named `PetPlayDateAI`.
2. Replace the generated Swift files with the files in this folder.
3. Keep `Assets.xcassets` from your Xcode project, or add custom colors if desired.
4. Run on iPhone simulator.

The UI is built entirely in SwiftUI and does not require any third-party dependencies.
